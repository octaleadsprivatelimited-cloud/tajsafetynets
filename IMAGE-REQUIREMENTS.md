## Image Specifications

- **Format**: JPG, PNG, or WebP
- **Recommended Size**: 1200x800 pixels (or similar aspect ratio)
- **File Size**: Optimized for web (under 500KB per image)
- **Quality**: High quality, professional photos showing the service/product

## Notes

- Some images already exist in the directory and can be reused
- All new images should be downloaded from the internet with appropriate service titles
- Ensure images are properly optimized for web use
- Images should be clear, professional, and relevant to each service
